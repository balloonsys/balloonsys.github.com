“在CentOS 6.5环境下安装Samba服务以共享文档”

Sep 21st, 2014

Samba是一套使用SMB(Server Message Block)协议的应用程序，通过支持这个协议，Samba允许Linux服务器与Windows系统之间进行通信。所以，采用Samba可以支持Mac、Windows系统访问CentOS服务器上的共享文档。

Samba服务器包括两个后台应用程序：Smbd和Nmbd。Smbd是Samba的核心，主要负责建立Linux Samba服务器与Samba客户机之间的对话，验证用户身份并提供对文件和打印系统的访问；Nmbd主要负责对外发布Linux Samba服务器可以提供的NetBIOS名称和浏览服务。

安装

yum install samba samba-client samba-common

查看版本

smbd —version

自动启动

chkconfig smb on chkconfig nmb on

防火墙配置

iptables -I INPUT 4 -m state —state NEW -m udp -p udp —dport 137 -j ACCEPT iptables -I INPUT 5 -m state —state NEW -m udp -p udp —dport 138 -j ACCEPT iptables -I INPUT 6 -m state —state NEW -m tcp -p tcp —dport 139 -j ACCEPT

创建共享目录，设置用户组及目录权限

mkdir -p /data/samba/doc

useradd sc groupadd smbgrp usermod -a -G smbgrp sc smbpasswd -a smbuser

chown -R sc:smbgrp doc/ chmod -R 777 /data/samba/doc

SAMBA配置

mv /etc/samba/smb.conf /etc/samba/smb.conf.bak vi /etc/samba/smb.conf

[global] workgroup = WORKGROUP ;security = share map to guest = bad user [Share] path = /data/samba/doc valid users = @smbgrp guest ok = no writable = yes browsable = yes

启动服务

service smb restart service nmb restart

访问共享目录

XP: \SERVER_IP\Share Mac: smb:\ SERVER_IP\Share
