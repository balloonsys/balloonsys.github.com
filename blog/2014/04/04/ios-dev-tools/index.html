做iOS外包开发用到的工具

Apr 4th, 2014

这个虚拟的Balloonsys工作室，总算发展到两位工程师了。

发展历史

2010年8月第一次为成都一家公司做兼职开发，仅持续两周 2012年9月帮杭州一家公司发布应用 2014年4月终于拿下了第一份APP开发订单，也找到了第一个合作开发工程师

第三方服务

为自己、为别人开发iPhone应用的过程中，也使用过不少第三方服务。一直持续使用的有：
1. 私有代码托管 https://bitbucket.org 2. 收集崩溃日志 https://www.crashlytics.com 3. 测试版本分发 https://testflightapp.com 4. 数据统计分析 http://www.appannie.com
